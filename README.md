# weather-app-dashboard

A responsive weather dashboard built with HTML, CSS and JavaScript using the Open-Meteo API.

## Features

- Search weather by city
- Fetch live weather data from an API
- Display temperature, condition, humidity, wind speed and feels-like temperature
- Weather condition icons
- Loading and error messages
- Recent search history
- Clickable recent city buttons
- LocalStorage for recent searches
- Dark/light mode
- Responsive layout

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
- Open-Meteo API
- LocalStorage
- GitHub Pages
